 Q_lr           左右動作

 h      N  h            左 (亦: CTRL-H、<BS> 或 <Left> 鍵)
 l      N  l            右 (亦: <Space> 或 <Right> 鍵)
 0         0            至本行第一個字符 (亦: <Home> 鍵)
 ^         ^            至本行第一個非空白字符
 $      N  $            至本行 (加上 N - 1 後續行) 最後一個字符 (亦: <End> 鍵)
 g0        g0           至屏幕行第一個字符 (當行迴繞時和 "0" 不同)
 g^        g^           至屏幕行第一個非空白字符 (當行迴繞時和 "^" 不同)
 g$     N  g$           至屏幕行最後一個字符 (當行迴繞時和 "$" 不同)
 gm        gm           至屏幕行中點
 bar    N  |            至第 N 列 (缺省: 1)
 f      N  f{char}      至右邊第 N 次出現 {char} 之處 (find)
 F      N  F{char}      至左邊第 N 次出現 {char} 之處 (Find)
 t      N  t{char}      至右邊第 N 次出現 {char} 之前 (till)
 T      N  T{char}      至左邊第 N 次出現 {char} 之前 (Till)
 ;      N  ;            重複上次 "f"、"F"、"t" 或 "T" 命令 N 次
 ,      N  ,            以相反方向重複上次 "f"、"F"、"t" 或 "T" 命令 N 次

 _      N  _            下移 N - 1 行，至第一個非空白字符
 G      N  G            至第 N 行 (缺省: 末行) 第一個非空白字符
 gg     N  gg           至第 N 行 (缺省: 首行) 第一個非空白字符
 N%     N  %            至全文件行數百分之 N 處；必須給出 N，否則是  %  命令
 gk     N  gk           上移 N 屏幕行 (迴繞行時不同於 "k")
 gj     N  gj           下移 N 屏幕行 (迴繞行時不同於 "j")

 Q_pa           模式查找

 /      N  /{pattern}[/[offset]]<CR>
                        向前查找第 N 次出現 {pattern} 的地方
 ?      N  ?{pattern}[?[offset]]<CR>
                        向後查找第 N 次出現 {pattern} 的地方
 /<CR>  N  /<CR>        向前重複上次查找
 ?<CR>  N  ?<CR>        向後重複上次查找
 n      N  n            重複上次查找
 N      N  N            相反方向重複上次查找
 star   N  *            向前查找光標下的標識符
 #      N  #            向後查找光標下的標識符
 gstar  N  g*           同 "*"，但也查找部分匹配
 g#     N  g#           同 "*"，但也查找部分匹配
 gd        gd           至光標下標識符的局部聲明 (goto declaration)
 gD        gD           至光標下標識符的全局聲明 (goto Declaration)

 pattern                查找模式中的特殊字符

                        含義          magic   nomagic   
                    匹配任意單個字符    .       \.
                            匹配行首    ^       ^
                          匹配 <EOL>    $       $
                        匹配單詞開始    \<      \<
                        匹配單詞結束    \>      \>
            匹配單個標定範圍中的字符    [a-z]   \[a-z]
        匹配單個不在標定範圍中的字符    [^a-z]  \[^a-z]
                  匹配一個標識符字符    \i      \i
                    同上但不包括數字    \I      \I
                  匹配一個關鍵字字符    \k      \k
                    同上但不包括數字    \K      \K
                  匹配一個文件名字符    \f      \f
                    同上但不包括數字    \F      \F
                匹配一個可顯示的字符    \p      \p
                    同上但不包括數字    \P      \P
                    匹配一個空白字符    \s      \s
                  匹配一個非空白字符    \S      \S

                          匹配 <Esc>    \e      \e
                          匹配 <Tab>    \t      \t
                           匹配 <CR>    \r      \r
                           匹配 <BS>    \b      \b

           匹配 0 或多個前面的匹配原    *       \*
           匹配 1 或多個前面的匹配原    \+      \+
          匹配 0 或 1 個前面的匹配原    \=      \=
          匹配 2 至 5 個前面的匹配原    \{2,5}  \{2,5}
                隔開兩種可替換的匹配    \|      \|
                組合模式為單個匹配原    \(\)    \(\)

 search-offset          查找命令後可用的偏移量

    [num]       向下 [num] 行，位於第一列
    +[num]      向下 [num] 行，位於第一列
    -[num]      向上 [num] 行，位於第一列
    e[+num]     匹配結尾 (end) 右方 [num] 個字符處
    e[-num]     匹配結尾 (end) 左方 [num] 個字符處
    s[+num]     匹配啟始 (start) 右方 [num] 個字符處
    s[-num]     匹配啟始 (start) 左方 [num] 個字符處
    b[+num]     同上 s[+num] (助記: begin)
    b[-num]     同上 s[-num] (助記: begin)
    ;{search-command}   接著執行 {search-command} 查找命令

 Q_ma           位置標記和相關動作

 m         m{a-zA-Z}    用標記 {a-zA-Z} 記錄當前位置

 `A        `{A-Z}       至任何文件中的標記 {A-Z}
 `0        `{0-9}       至 Vim 上次退出的位置

 `quote    `"           至上次編輯此文件的位置
 `[        `[           至上次被操作或放置的文本的開始
 `]        `]           至上次被操作或放置的文本的結尾
 `<        `<           至 (前次) 可視區域的開始
 `>        `>           至 (前次) 可視區域的結尾
 `.        `.           至當前文件最後被改動的位置
 '         '{a-zA-Z0-9[]'"<>.}
                        同 `，但同時移動至該行的第一個非空白字符
 :marks   :marks        列出活動的標記
 CTRL-O   N  CTRL-O     跳轉到跳轉表中第 N 個較早的位置

 :ju      :ju[mps]      列出跳轉表
